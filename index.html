Lineage II • OBS/OCR • Online RB Map

Raid Boss timers and shared online map for Lineage II.

Yura RB Monitor reads visible system chat through OBS/OCR, detects Raid Boss messages, starts respawn timers, shows an in-game overlay and synchronizes active RB timers on a shared online map.

Trial and paid license keys are handled by the launcher. The client is distributed as a Windows ZIP package.

What it does

Built for Lineage II Raid Boss monitoring with shared timers for clan or party usage.

OBS/OCR chat detection

Reads the configured chat capture zone and detects RB respawn messages without controlling your character.

In-game overlay

Shows active RB timers directly over the game window with configurable position, color and scale.

Shared online map

When one user detects a boss, every opened map can see the same active timer and boss marker.

Launcher licensing

Trial activation, license keys, status checks and a clean launcher UI for normal users.

Search and preview

Search an RB to quickly preview its location; active timers remain visible globally.

High Five RB workflow

Designed around Lineage II RB messages, respawn timers and practical map navigation.

How to install

Download the client ZIP.
Get the latest Windows package from GitHub Releases.
Extract the ZIP.
Do not run the EXE from inside the archive. Extract the full folder first.
Run YuraRB_Launcher.exe.
Activate trial or enter your license key.
Configure OBS and overlay.
Select the OBS source and chat capture zone, then start monitoring.
