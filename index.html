What it does

Designed for quick Raid Boss monitoring without controlling your character or modifying the game client.

👁

OBS/OCR detection

Reads visible system chat through OBS screenshots and detects Raid Boss messages.

⏱

Respawn timers

Starts a timer when a boss appears in chat and keeps it visible in the overlay.

🗺

Online RB map

Shows only active timer bosses by default. Search and “show all RB” modes help locate spots.

🌐

Shared data

If one user detects a boss, all users with the map can see the active timer.

🔐

License launcher

Trial and paid key activation are handled by the Yura RB launcher and license server.

🧭

Search & focus

Search RB by English or Russian name and focus the map on the boss location.

Quick setup

Basic client flow. Detailed OBS setup can be added later.

Download

Download the ZIP from the latest release and unpack it into a separate folder.

Launch

Start YuraRB_Launcher.exe. Do not run the monitor EXE alone.

Activate

Use trial mode or enter your paid license key.

Play

Run OBS capture and start the monitor. Detected bosses will appear on the map.
